# Protocol Hub

Welcome to the **Protocol Hub** — comprehensive documentation for communication protocols used in fleet management and telematics systems.

## Overview

The Protocol Hub documents the communication protocols used between GPS devices, vehicle systems, and fleet management platforms. This includes standard protocols, proprietary protocols, and emerging standards.

## Protocol Categories

### Standard GPS Protocols

Widely adopted protocols supported by multiple device manufacturers.

- **GT06 / GT02** - Popular protocol for Chinese GPS trackers
- **TK103** - Common protocol with many variants
- **H02** - Protocol used by many entry-level devices
- **Meitrack** - Protocol for Meitrack devices

### Proprietary Protocols

Vendor-specific protocols requiring custom implementation.

- **Teltonika** - Binary protocol with codec support
- **CalAmp** - Protocol for CalAmp devices
- **Syrus** - Syrus device protocol

### Vehicle Communication Protocols

Protocols for communicating with vehicle systems.

- **OBD-II** - On-board diagnostics standard
- **CAN Bus** - Controller Area Network
- **J1939** - Heavy-duty vehicle communication
- **J1708** - Older heavy-duty vehicle protocol

### Network Protocols

Protocols for data transmission over networks.

- **HTTP/REST** - Web API communication
- **MQTT** - Lightweight messaging for IoT
- **WebSocket** - Real-time bidirectional communication
- **TCP/UDP** - Transport layer protocols
