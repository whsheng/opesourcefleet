# Software Hub

Welcome to the **Software Hub** — a comprehensive repository of open-source fleet management software solutions.

## Overview

The Software Hub catalogs and categorizes various open-source software solutions designed for fleet management, telematics, GPS tracking, and related transportation technologies.

## Categories

### Fleet Management Platforms

Complete solutions for managing vehicle fleets, including real-time tracking, maintenance scheduling, and route optimization.

| Software | Language | License | Description |
|----------|----------|---------|-------------|
| [Traccar](https://www.traccar.org/) | Java | Apache 2.0 | Modern GPS tracking platform |
| [OpenFMS](https://openfms.io) | Go/React | Apache 2.0 | Modern fleet management system |

### GPS Tracking & Telematics

Software focused on GPS tracking, vehicle diagnostics, and telematics data collection.

### Route Optimization

Solutions for optimizing delivery routes, reducing fuel consumption, and improving efficiency.

### Maintenance Management

Tools for scheduling preventive maintenance, tracking repairs, and managing vehicle lifecycles.

### Driver Management

Software for managing driver information, certifications, behavior monitoring, and compliance.

## How to Contribute

If you know of an open-source fleet management software that should be listed here, please submit a pull request or open an issue on our GitHub repository.

## Quick Start

Browse software by category using the sidebar, or use the search function to find specific solutions.
