# Fleet Management Platforms

Complete fleet management platforms that provide end-to-end solutions for vehicle fleet operations.

## Featured Projects

### Traccar

**Traccar** is a free and open-source modern GPS tracking platform.

- **Website**: [traccar.org](https://www.traccar.org/)
- **GitHub**: [traccar/traccar](https://github.com/traccar/traccar)
- **Language**: Java
- **License**: Apache License 2.0

**Key Features:**
- Real-time GPS tracking
- Support for 2000+ GPS protocols
- Web and mobile applications
- Geofencing and alerts
- Reports and analytics
- REST API

---

### OpenFMS

**OpenFMS** is a modern fleet management system built with Go and React.

- **Website**: [openfms.io](https://openfms.io)
- **GitHub**: Coming soon
- **Language**: Go, React, TypeScript
- **License**: Apache License 2.0

**Key Features:**
- Modern architecture (Go + React)
- Real-time tracking
- Scalable design
- RESTful API
- Multi-tenant support
